算法1-几种简单的排序算法

d 希尔排序

简单插入排序在排序过程中太过老实了,只是不断的做简单的插入.例如序列[5,4,3,2,1,0]中靠后的序列想要将0放到第一个位置上的时候需要经过大量的排序,复杂度十分的高,因此希尔在59年提出了希尔排序,这是第一种复杂度冲进o(n^2)的排序算法.

下面简单讲解一下算法的主要思想:

算法主要是通过跨越的方式使得数字尽可能快的接近自己的正确位置.那么数字如何更快的到达自己的位置呢?

这里就需要介绍一个概念了:增量序列.增量序列指的是一个不断递减的序列.以希尔增量:{n/2,(n/2)/2,….}为例.第一个数字n/2指的是待排序数列中每隔n/2个位置的数作为一组进行插入排序.在排序完成后使用增量的第二个数进行排序,发现没有随着排序的进行,增量在减小、排序的组数在减少、每组中排序的数量在增加.

在排序刚开始的时候跨度往往很大,就是由于这种大跨度的排序,使得数字可以更快的到达自己应该处于的位置.

希尔排序中一个最难的问题就是如何选择增量序列,增量序列的选择是个数学上的难题.希尔增量并不是最好的增量序列,目前使用较多的增量序列是Hibbard,最坏的时间复杂度是o(n^3/2).

下面贴出基于希尔增量实现的算法源码,(用户想要更加优秀的性能可以自己更换想要的增量序列)

public class ShellSort{
    public static void main(String[] args){
        System.out.println("hello");
        int[] nums = {
            4,6,6,5,3,2,4,6,78,9,7,6,5,4,43,32,2
        };
        Shell(nums);
        for(int i = 0; i < nums.length; i++){
            System.out.println(nums[i]);
        }
    }
    public static void Shell(int[] nums){
        int length = nums.length/2;
        while(length > 0){
            // 依据length进行插入排序
            for(int i = 1; i < nums.length; i++){
                int x = i;
                while(x >= 0 && x - length >=0){
                    if(nums[x] >= nums[x - length]){
                        break;
                    }else{
                        int t = nums[x];
                        nums[x] = nums[x - length];
                        nums[x - length] = t;
                        x -= length;
                    }
                }
            }
            length /= 2;
        }
    }
}

e 归并算法

归并算法主要利用了分治的思想,将一个待排序的数组左右平分为两份,通过分左右的思想一直向下分,当都分为一个的时候就是有序的.然后再将左右两个有序的数组进行合二为一.最终就得到了有序的数组.

算法二分的深度为lgn,并且每次比较的复杂度为n因此总的时间复杂度为o(nlgn),而空间复杂度为o(n),因为两个待排序的数组都需要创建一个空的同等长度的数组进行存放.因此这是一个稳定的、时间复杂度为o(nlgn),空间复杂度为o(n)的算法

public class MergeSort{
    public static void main(String[] args){
        System.out.println("h");
        int[] nums = {
            2,4,6,6,5,4,3,4,7,8,7,5,4,3,2,2,4,567,4,2,765
        };
        //Merge1(nums,0,nums.length-1);
        Merge2(nums);
        for(int i = 0; i < nums.length; i++){
            System.out.println(nums[i]);
        }
    }
    // 非递归
    public static void Merge2(int[] nums){
        int len = 2;
        while (len < 2 * nums.length){
            int startN = 0;
            while(startN < nums.length){
                int overN;
                if (startN + len < nums.length) overN = startN + len - 1;
                else overN = nums.length - 1;
                if (overN - startN + 1 < len / 2) {
                    break;
                }
                int mid = startN + len  / 2;
                
                int[] t = new int [nums.length];
                for(int i = 0; i < nums.length; i++){
                    t[i] = nums[i];
                }
                
                int lo = startN;
                int hi = overN;
                int m = mid;
                
                int x = lo; 
                while(lo < m || mid < hi +1 ){
                    if (lo >= m){
                        nums[x] = t[mid];
                        mid++;
                        x++;
                    }else if(mid >= hi + 1) {
                        nums[x] = t[lo];
                        lo++;
                        x++;
                    }else if (lo < m && t[lo] < t[mid]){
                        nums[x] = t[lo];
                        lo++;
                        x++;
                    }else if (t[lo] >= t[mid] && mid < hi + 1){
                        nums[x] = t[mid];
                        mid++;
                        x++;
                    }
                }
                startN += len;
            }

            len *= 2;
        }
    }
    // 递归
    public static void Merge1(int[] nums, int lo, int hi){
        if (lo == hi){
            return;
        }else{
            int mid = (lo + hi) / 2;
            Merge1(nums, lo, mid);
            Merge1(nums, mid+1, hi);
            int[] t = new int[nums.length];
            for(int i = 0; i < nums.length; i++){
                t[i] = nums[i];
            }
            int m = mid + 1;
            mid++;
            int x = lo;
            while(lo < m || mid < hi +1 ){
                if (lo >= m){
                    nums[x] = t[mid];
                    mid++;
                    x++;
                }else if(mid >= hi + 1) {
                    nums[x] = t[lo];
                    lo++;
                    x++;
                }else if (lo < m && t[lo] < t[mid]){
                    nums[x] = t[lo];
                    lo++;
                    x++;
                }else if (t[lo] >= t[mid] && mid < hi + 1){
                    nums[x] = t[mid];
                    mid++;
                    x++;
                }
            }
        }
    }
}

f 原地归并算法

归并算法的时间复杂度非常优秀,不论待排序数组的情况如何,所话费的时间都是nlgn,但是空间复杂度达到了o(n),这也是这个算法最大的缺点,因此通过改进算法实现就得到了原地归并算法,空间复杂度为o(1)

思想:

原地归并的的复杂度相对于非原地归并来说会上升一点点局部一次的复杂度为o(2n)，但是相较于使用的空间来说是很大的进步了。

非原地归并解发的问题就是如何实现在原地的掉头问题, 首先找到第一个比第二个序列中第一个数大的数，然后在第二个序列中找到最后一个比上一个找到的数小的数，这时候就需要在原地调转了，前面的部分做一次旋转，后面的也做一次旋转，整体再来一次就实现了。

具体实现：可以直接用上面的方法体进行调用

    public static void  Merge3(int[] nums, int lo, int hi){
        if (lo == hi){
            return;
        }else{
            int mid = (lo + hi) / 2;
            Merge3(nums, lo, mid);
            Merge3(nums, mid+1 , hi);
            int m = mid + 1;
            while(lo <=  hi || m <= hi){
                while(nums[lo] <= nums[m] && lo < m) lo++;
                if (lo != m){
                    int m1 = m;
                    while(m + 1 <= hi && nums[m+1] < nums[lo]) m++;
                    //  原地倒转 lo-m1 m1-m
                    for(int i = lo; i <= (lo + m1 -1)/2; i++){
                        int t = nums[i];
                        nums[i] = nums[lo + m1 -1 -i];
                        nums[lo + m1 - 1 - i] = t;
                    }
                    for (int i = m1; i <= (m1 + hi) / 2; i++){
                        int t = nums[i];
                        nums[i] = nums[m1 + hi - i];
                        nums[m1 + hi - i] = t;
                    }
                    for (int i = lo; i <= (lo + hi) / 2; i++ ){
                        int t = nums[i];
                        nums[i] = nums[lo + hi -i];
                        nums[lo + hi - i] = t;
                    }
                }else{
                    break;
                }
            }
        }
    }

g 快速排序

快速排序还是很简单的，弄清楚原理之后就会发现也不过如此。主要的实现就是通过找到当前第一个元素在数组中的正确的位置，并且该元素找到位置之后，对该元素前面的数组继续使用该方法，后面也是，最终所有元素找到自己的位置即可。

具体实现：

public class QuickSort{
    public static void main(String[] args){
        System.out.println("j");
        int[] nums = {
            1,4,6,8,7,6,56,6,7,5,43,5,7,43,2,36,8,65,3,456,89,76,543,342,6,54,4,43
        };
        quick(nums,0,nums.length - 1);
        for(int i = 0; i < nums.length; i++){
            System.out.println(nums[i]);
        }
    }
    public static void quick(int[] nums, int lo, int hi){
        int start = lo;
        int over = hi;
        int j=lo;
        while (lo < hi){
            if (j == lo && nums[hi] >= nums[lo]){
                hi--;
            }else if (j == hi && nums[hi] >= nums[lo]){
                lo++;
            }else if (j == lo && nums[hi] < nums[lo]){
                change(nums, lo, hi);
                j = hi;
            }else{
                change(nums, lo, hi);
                j = lo;
            }
        }
        if (start < lo - 1) quick(nums,start,lo-1);
        if (over > lo + 1) quick(nums, lo + 1, over);
    }
    public static void change(int[] nums, int index1,int index2){
        int t = nums[index1];
        nums[index1] = nums[index2];
        nums[index2] = t;
    }
}

h 计数排序

public class CountingSort{
    public static void main(String[] args){
        System.out.println("a");
        int[] nums = {
            2,4,6,4,3,3,4,5,7,8,2,5,4,3,2,3,4,7,9,8,3,6,4
        };
        Counting(nums,0,10);
        for(int i = 0; i<nums.length; i++){
            System.out.println(nums[i]);
        }
    }
    // 计数排序适合已经明确待排序范围，并且范围不是很大的算法，时间复杂度是o(n+k)，其中n表示个数，k表示范围
    public static void Counting(int[] nums, int min, int max){
        int[] k = new int[max - min + 1];
        for (int i = 0; i<k.length;i++){
            k[i]=0;
        }
        for(int i = 0; i < nums.length; i++){
            k[nums[i] - min]++;
        }
        int t = 0;
        for(int i = 0; i < k.length; i++){
            for(int j = 0; j < k[i]; j++){
                nums[t] = min + i;
                t++;
            }
        }
    }
}
