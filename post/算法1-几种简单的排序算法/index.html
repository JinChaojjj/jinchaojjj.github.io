算法1-几种简单的排序算法

本文将介绍几种简单的排序算法,下文首先会介绍一下几种算法的基本概念和思想,然后给出用java实现的

a 冒泡排序

冒泡排序的思想是从数组的头开始进行相邻两个数之间的比较,将较大的数换到后边,然后一次向下

一轮的比较将选出待比较数中最大的,有点像最大的泡泡飘在最上面的感觉因此叫冒泡排序

代码如下:

public class BubbleSortTest{
    public static void main(String[] args){
        int[] nums = {
            8,7,5,4,4,5,6,4,2,1
        };
        BubbleSort(nums);
        for(int i = 0; i < nums.length; i++){
            System.out.println(nums[i]);
        }
    }
    public static void BubbleSort(int[] nums){
        for(int i = 0; i < nums.length; i++){
            for(int j = nums.length - 1; j > i; j--){
                if (nums[j] < nums[j-1]){
                    int t = nums[j];
                    nums[j] = nums[j-1];
                    nums[j-1] = t;
                }
            }
        }
    }
}

b 选择排序

选择排序指的是在队列中找到最大的数所在的下标,之间与最后一个数进行交换位置,以此类推

代码如下:

public class selectionSort{
    public static void main(String[] args){
        int[] nums = {
            1,3,4,5,4,3,4,5,4,3,8,7,5
        };
        selection(nums);
        for(int i=0; i< nums.length; i++){
            System.out.println(nums[i]);
        }
        System.out.println("Hello");
    }
    public static void selection(int[] nums){
        for(int i = nums.length -1; i > 0; i--){
            int t = 0;
            for(int j = 0; j <= i; j++){
                if (nums[j] > nums[t]) t=j;
            }
            int tt = nums[t];
            nums[t] = nums[i];
            nums[i] = tt;
        }
    }
}

c 插入排序

插入排序指的是假设前m个是已经有序的数列,将第m+1个插入前方已经有序的数列中正确的位置,这种方法在基本有序的队列中效率十分的高

代码如下:

public class InsertionSort{
    public static void main(String[] args){
        System.out.println("hello");
        int[] nums = {
            1,8,7,5,4,3,4,5,6,67,5,3,2,1
        };
        insertion(nums);
        for(int i = 0; i < nums.length; i++){
            System.out.println(nums[i]);
        }
    }
    public static void insertion(int[] nums){
        for(int i = 1; i < nums.length; i++){
            for (int j = i; j > 0; j--){
                if(nums[j] < nums[j-1]){
                    int t = nums[j];
                    nums[j] = nums[j-1];
                    nums[j-1] = t;
                }
            }
        }
    }
}
