a、链表

链表主要分为单链表、双链表、循环链表

其中单链表指的是单向的链表，一般是有一个头结点、并且方向是单一的，从头指向尾。

双向链表指的是，每一个节点都有指向前序节点和后续节点的指针，从任何位置都可以向前向后走

循环链表指的是收尾相互连接的链表

单链表结构

class Linked{
    private Integer val;
    private Linked next;

    public Linked(Integer value){
        this.val = value;
        this.next = null;
    }

    public boolean haveNext(){
        if (next == null) return false;
        else return true;
    }

    public Linked getNext(){
        return this.next;
    }

    public void setValue(Integer value){
        this.val = value;
    }

    public int getValue(){
        return this.val;
    }

    public void setNext(Linked next){
        this.next = next;
    }
}

b、邻接表

邻接表的结构类似于创建一个节点的数组，数组中记录了场景中所有的节点，并且每个节点可以记录该节点连接的所有的线。

其实还可以理解为多个链表的合集，将所有的头结点都存放在一个数组中。

邻接表结构

单一的节点还是使用在链表中提到的Linked

Linked[] linkeds = new Linked(n);// 若该邻接表记录的是图，则n表示这个图的节点数

添加节点

public Linked addNode(x){
    Linked[] newLin = new Linked(n + 1);
    for (int i = 0; i < n; i++){
        newLin[i] = linkeds[i];
    }
    linkeds[n] = new Linked();
    linkeds[n].setVal(x);
    linkeds = newLin;
    return linkeds[n];
}

插入边

public void addL(int x, int y){
    Linked p;
    for (int i = 0; i < linkeds.length; i++){
        if (linkeds[i].getVal() == x) p = linkeds[i];
    }
    if (p == null) p = addNode(x);
    Linked q;
    q = p;
    while(q != null){
        if (q.getNext() == null)  q.setNext(new Linked(y););
        else if (q.getNext().getVal() == y) break; // 本身存在这条边
        else q = q.getNext();
    }
}

public void createTree (int[] iList,Linked node, int n){
    if (2 * n + 1 < iList.length){
        Linked left = new Linked(iList[2 * n + 1]);
        node.setLeft(left);
        createTree(iList, left, 2 * n +1);

    if (2 * n + 2 < iList.length){

        node.setRight(right);
        createTree(iList, right, 2 * n +2);
    }
}

f、kmp

public class SubstringSearch {
    //构建失配数组
    private int[] computeTemporaryArray(char pattern[]){
        int [] lps = new int[pattern.length];
        int j =0;
        for(int i=1; i < pattern.length;){
            if(pattern[i] == pattern[j]){
                lps[i] = j + 1;
                j++;
                i++;
            }else{
                if(j != 0){
                    j= lps[j-1];
                }else{
                    lps[i] = 0;
                    i++;
                }
            }
        }
        return lps;
    }

    //KMP匹配
    public boolean KMP(char []text, char []pattern){
        int lps[] = computeTemporaryArray(pattern);
        int i=0;
        int j=0;
        while(i < text.length && j < pattern.length){
            if(text[i] == pattern[j]){
                i++;
                j++;
            }else{
                if(j!=0){
                    j = lps[j-1];
                }else{
                    i++;
                }
            }
        }
        if(j == pattern.length){
            return true;
        }
        return false;
    }

    public static void main(String args[]){
        String str = "abcxabcdabcdabcy";
        String subString = "abcdabcy";
        SubstringSearch ss = new SubstringSearch();
        boolean result = ss.KMP(str.toCharArray(), subString.toCharArray());
        System.out.print(result);
    }
}

kmp算法中最重要的就是计算next数组，也就是上面版子里面的computeTemporaryArray方法，通过计算next数组，可以很快确认，字符串中访问到的当前字符位置中，与字符串最大的重合前缀，就可以不用对待找字符串进行回溯查找，实现了o(n)的时间复杂度。

          	return;

          	if (p.getVal() = num) return;
          	else if (p.getVal() > num){

                }

        }
    }
