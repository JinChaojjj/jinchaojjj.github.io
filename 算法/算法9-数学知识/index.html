算法9-数学知识

1.质数

1.1 质数的定义

质数定义：若整数n是质数，则在整数1-n中n只能被1和n整除，其余所有的数都无法整除

1.2 判断n是否为质数

方法1:判断1-n，时间复杂度o(n)

public boolean isPrimeNumber(int n){
		boolean flag = false;
  	for (int i = 2; i < n; i++){
				if (n % i == 0){
          	flag = true;
          	break;
        }
    }
  	return flag;
}

该方法判断1-n之间所有的数，若存在余数为0的数，则表示n不是质数，若不存在则是质数。

但是这个方法存在一个问题，就是所有大于sqrt(n)的数，如果是n的因子，那么一定存在一个小于sqrt(n)的数同样也是n的因子，上述方法在判断非质数时不存在这个问题，小的因子在初期就已经跳出循环，但是对于质数的判断还可以优化，将复杂度控制为o(sqrt(n))。

方法2:o(sqrt(n))

public boolean isPrimeNumber(int n){
		boolean flag = false;
  	for (int i = 2; i < (int) Math.sqrt(n); i++){
				if (n % i == 0){
          	flag = true;
          	break;
        }
    }
  	return flag;
}

1.3 打印质数表

方法1:使用1.2方法 o(nsqrt(n))

该方法的思路是，需要得到1-n之间所有质数的列表，那就将区间中所有的数都按照1.2方法2进行一次判断

方法2:埃氏筛法 o(nloglogn)

思路：求取2-n中所有质数，埃氏筛法的思想就是从小到大开始便利，设当前便利到x，则将小于n的所有倍数标记为false/-1，然后继续向下便利没有被标记的数。

因为是从小到大开始便利，便利到当前数则表示比当前数小的所有数都不是当前数的因子，因此可以筛选出来

public List<int> primeNumberInN(int n){
		int[] nu = new int[n + 1];
  	for (int i = 0; i <= n; i++){
      	nu[i] = 0;
    }
  	nu[0] = -1;
  	nu[1] = -1;
  	for(int i = 2; i <= n; i++){
      	if (nu[i] == 0){
          	int j = i * 2;
          	while (j <= n){
              	nu[i] = -1;
            }
        }
    }
  	List<int> ans = new ArrayList<int>();
  	for(int i = 2; i <= n; i++){
      	if (nu[i] == 0) ans.add(i);
    }
  	return ans;
}

方法3:欧拉筛法 o(n)

如果i是质数，那么就将它与之前的质数（包括它本身）的乘积筛掉 。

如果i是合数，那么就将它与从2到它最小的质因子之间的质数的乘积分别筛掉。

public List<int> primeNumberInN(int n){
  	List<int> ans = new ArrayList<int>():
  	int[] nu = new int[n + 1];
  	for (int i = 0; i <= n; i++){
      	nu[i] = 0;
    }
  	for (int i = 2; i <= n; i++){
      	if (nu[i] == 0){
          	ans.add(i);
          	foreach(int j in ans){
              	nu[i * j] = -1;
            }
        }else{
          	foreach(int j in ans){
              	nu[i * j] = -1;
              	if (i % j == 0) break;
            }
        }
    }
  	return ans;
}

2.约数

2.1 最大公约数

public int gcd(int a, int b){
		if(b == 0) return a;
  	else return gcd(b, a % b);
}

2.2 最小公倍数

public int lcm(int a, int b){
  	return a * b / gcd(a, b);
}

3.欧拉函数

欧拉函数的求解的是对于整数n，所有小于n且与n互至的数的个数

公式为ϕ(n)=n(1−1/p1)(1−1/p2)(1−1/p3)….(1−1/pn)

public int oula(int n){
  	int ans = n;
  	for(int i = 2; i < (int) Math.sqrt(n));i++){
      	if (n % i == 0){
          	ans = ans / i * (i - 1);
          	while(n % i == 0){
              	n /= i;
            }
        }
    }
  	if (n > 1) ans = ans / n * (n - 1);
  	return ans;
}

也可以使用欧拉筛来计算，就是使用1.3中的方法3计算得到欧拉筛，然后利用公式计算就可以。

两种方法的复杂度都是o(n)，计算当个数的欧拉值区别不大，但是在计算多个数的欧拉值的时候，使用欧拉筛会快很多

4.快速幂

递归快速幂

// a的b次方
public int fastPower(int a, int b){
  	if (b == 0) return 1;
  	if (b == 1) return a;
  	if (b == 2) return a * a;
  	return fastPower(a, b / 2) * fastPower(a, b / 2) * fastPower(a, b % 2);
}
